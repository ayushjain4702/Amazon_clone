# Amazon-clone using ReactJS
Project Link - https://amazon-clone-4702.netlify.app/

**This is a clone of popular e-commmerce website Amazon. I have created it with ReactJS**

## Tech Used:
- ReactJs
- Firebase hosting
- Firebase Auth
- Firebase Storage
- React-Dom
- React-Redux

## Features:
- Login and Logout
- Add to cart
- Remove from cart
- Neat and Clean Ui
